Building JManage


Preparing
  • JManage requires JDK 1.4+ to build and run. You can download JDK here.
  • Ant 1.6.1 is used to build JManage. You can download ant here.
  • Get the JManage source tree from cvs.sourceforge.net. Windows developer can setup WinCVS as described in this document.
    <jmanage-home> in this document refers to c:\cvsroot\main, where c:\cvsroot is you CVS root.
  • JManage in this initial development stage, only supports Weblogic 6.1 and requires Weblogic 6.1 to build.
    After you install Weblogic 6.1, you must copy weblogic.jar to <jmanage-home>/ext folder (you will need to create this folder).
    The weblogic.jar should be renamed to weblogic61.jar.
    [This dependency will be removed before the first release.]

Building
  • Make sure that JAVA_HOME and ANT_HOME environment variables are properly set.
  • Run "ant" from <jmanage-home> directory to create the build.

Running JManage
  • Execute startup.cmd from <jmanage-home>. JManage runs on port 9090 by default. JManage homepage:
    http://localhost:9090/index.jsp
    (We don't yet have a startup.sh for unix developers. Please add this script if you are developing on unix environment.)
