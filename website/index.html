JManage Overview


JManage is a web based application management software, which provides a centralized console for managing application clusters and distributed application production environments.

We are currently developing version 1.0, which will have the following features:
  • Support for different J2EE app servers
  • Authentication and Role Management
  • Application configuration management
  • Runtime Graphing
  • Alerts
  • JMX notifications
  • Command line interface
